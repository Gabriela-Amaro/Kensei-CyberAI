# Kensei CyberAI Playground

## Saudações

Olá, viajante de dados! Este espaço nasceu para acompanhar os estudos do curso Kensei CyberAI e reunir experimentos que misturam segurança ofensiva, defesa e inteligência artificial.

## Propósito

- Consolidar aprendizados do treinamento da Kensei sobre CyberAI.
- Registrar laboratórios, scripts e insights de segurança ofensiva/defensiva.
- Criar artefatos visuais e práticos que conectam IA e cibersegurança.

## Primeira semana - Vibe Coding

Na primeira semana, foram criados scripts simples usando IA para praticar lógica em Python, manipulação de arquivos, menus interativos, contagem de palavras, geração de senhas, organização de arquivos, quiz com tempo e um scanner de portas básico.

Para ver o resumo detalhado dos códigos e o que cada script faz, acesse o arquivo da semana em [vibe_coding/semana_1.md](vibe_coding/semana_1.md).
